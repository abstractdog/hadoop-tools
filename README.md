# hadoop-tools

Simple tool to download a sample csv and load into hive.
Original data source: https://www.sample-videos.com/csv/

Usage:
1. Clone this repo
```
git clone https://github.com/abstractdog/hadoop-tools.git
```
2. Init it
```
source hadoop-tools/init.sh
```
3. Use it  
```
hdfs-shell.sh
```
